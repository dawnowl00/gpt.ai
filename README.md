# ONDAM AI

## 소개

ONDAM AI는 어르신의 정서 케어와 일상 관리를 지원하는 AI 동반자 플랫폼입니다. 사용자의 대화를 기억하고 감정을 분석하며, 보호자와의 연결을 통해 보다 안전하고 따뜻한 일상을 제공합니다.

## 주요 기능

* AI 음성 대화 및 정서 케어
* 사용자 맞춤 기억 기능
* 감정 분석 및 일상 리포트
* 보호자 앱 연동
* 복약 및 일정 알림
* SOS 긴급 호출 기능
* 위치 확인 및 안전 관리
* 이상 행동 감지 (Family Care)
* 홈캠 연동 모니터링

## 사용 방법

1. 저장소를 클론합니다.
2. 필요한 패키지를 설치합니다.
3. 환경 변수를 설정합니다.
4. 애플리케이션을 실행합니다.
5. 모바일 앱 또는 웹 서비스와 연동하여 사용합니다.

## 기술 스택

* Python
* FastAPI
* OpenAI API
* SQLite / PostgreSQL
* Flutter
* Bluetooth LE (BLE)

## 프로젝트 목표

가족이 항상 곁에 있을 수 없는 상황에서, 어르신의 외로움을 줄이고 정서적 안정과 안전한 일상을 지원하는 AI 동반자를 만드는 것을 목표로 합니다.
